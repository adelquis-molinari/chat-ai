## Getting Started

First, run the development server:


+ create an api key in [openai](https://openai.com)
+ create file **.env**
  + OPENAI_API_KEY = "your api key"


```bash
npm install

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
